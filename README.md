## Config files

settings.json:
Create this file in each project you want locked down
```
C:\path\to\your-project\.claude\settings.json
```


## Building flow

Personal productivity setup (Independent of any project):
/start (once), /update (regularly)	

Building a thing (Per project/feature):	
/product-brainstorming → /write-spec → /system-design → talk to Claude to build	

## House-building analogy (building flow)

| Step | Output | Who reads it |
|---|---|---|
| Brainstorm | "We need a place to live near my job" | You + your partner |
| Spec (PRD) | "3-bedroom family home, kitchen island, backyard, budget $500k, by August" | Homeowner ↔ architect |
| System design | "Load-bearing wall here, plumbing routed this way, HVAC zones A/B" | Architect ↔ builder |
| Build | Actual construction | Builder |
